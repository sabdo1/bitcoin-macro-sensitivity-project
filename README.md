# Bitcoin's Economic Integration

Research Question: Has Bitcoin's return behaviour become increasingly 
driven by macroeconomic factors in the post-COVID and monetary tightening period?

## Data Sources
- Bitcoin, S&P 500, VIX, DXY: Yahoo Finance
- 10Y Treasury Yield, Breakeven Inflation: FRED

## Sample Period
January 2017 – March 2026

## Methodology
1. Rolling window regressions
2. Structural break tests (Chow, Bai-Perron)
3. VAR model with IRFs and FEVD
4. Granger causality tests

## How to Run
Install dependencies: pip install -r requirements.txt
Run scripts in order: 01 then 02 then 03 then 04 then 05
